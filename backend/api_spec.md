# 記事共有アプリ API仕様書

## 共通設定

- **ベースURL:** `http://localhost:8000`
- **データ形式:** JSON

## 1. 非認証関連

### 記事一覧取得

全てのユーザーが投稿した記事を取得します。

- **メソッド:** `GET`
- **エンドポイント:** `/posts`

- **レスポンスボディ:**
  ```json
  [
    {
      "id": 1,
      "url": "[https://example.com](https://example.com)",
      "memo": "この記事は参考になります！",
      "username": "tanaka",
      "created_at": "2024-05-20T10:00:00"
    }
  ]
  ```

## 2. 認証関連

### 新規投稿

URLとおすすめ理由を投稿します。

- **メソッド:** `POST`
- **エンドポイント:** `/posts`

- **リクエストボディ:**
  ```json
  [
    {
      "url": "[https://example.com](https://example.com)",
      "memo": "おすすめの理由"
    }
  ]
  ```

### 特定記事1件取得

指定された記事IDの記事を1件取得する。
ただし、認証済みユーザー本人が投稿した記事のみ取得可能。

- **メソッド:** `GET`
- **エンドポイント:** `/posts/{post_id}`

- **レスポンスボディ:**

  ```json
  [
    {
      "id": 1,
      "title": "記事タイトル",
      "content": "記事内容",
      "user_id": 123,
      "created_at": "2026-03-19T12:00:00"
    }
  ]
  ```

### 投稿編集

指定された記事を更新する。
ただし、ログインユーザー本人が投稿した記事のみ更新可能。

- **メソッド:** `PUT`
- **エンドポイント:** `/posts/{post_id}`

- **リクエストボディ:**
  ```json
  [
    {
      "url": "https://example.com",
      "memo": "更新後のメモ内容"
    }
  ]
  ```
- **レスポンスボディ:**
  ```json
  [
    {
      "id": 1,
      "url": "https://example.com",
      "memo": "更新後のメモ内容",
      "user_id": 123,
      "created_at": "2026-03-19T12:00:00"
    }
  ]
  ```

### ログインユーザーの投稿記事全取得

ログインしているユーザー自身が投稿した記事一覧を取得する（マイページ用）

- **メソッド:** `GET`
- **エンドポイント:** `/my-posts`

- **レスポンスボディ:**

  ```json
  [
    {
      "id": 1,
      "title": "記事タイトル1",
      "content": "記事内容1",
      "user_id": 123,
      "created_at": "2026-03-19T12:00:00"
    },
    {
      "id": 2,
      "title": "記事タイトル2",
      "content": "記事内容2",
      "user_id": 123,
      "created_at": "2026-03-18T10:00:00"
    }
  ]
  ```

### 投稿削除

自分の投稿を削除します。

- **メソッド:** `DELETE`
- **エンドポイント:** `/posts/{post_id}`

### ログイン

メールアドレスとパスワードでログインし、認証用のJWTトークンを取得する。

- **メソッド:** `POST`
- **エンドポイント:** `/login`
- **レスポンスボディ:**トークンを返却します。

- **リクエストボディ:**

  ```json
  [
    {
      "email": "user@example.com",
      "password": "password123"
    }
  ]
  ```

- **レスポンスボディ:**
  ```json
  [
    {
      "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
      "token_type": "bearer"
    }
  ]
  ```
